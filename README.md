# javascript 数据结构与算法

欢迎查看博客园：[weiqinl.com](http://www.cnblogs.com/weiqinl/)
## 如何使用本库

```
git clone git@github.com:weiqinl/javascript-datastructures-algorithms.git & cd javascript-datastructures  
npm install  
npm test  
```

## 目录结构

```
src  
  algorithms(算法应用)  
  data-structures(数据结构)  
    __test__ 测试用例
```
